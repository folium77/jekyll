週間予報はあとにも先にも雨ばかりだったけれど、雨雲のほころびみたいに一日だけ晴れた梅雨の合間。前日からずっと本を読んでいたせいで脳にながれ込んでくる活字はしだいに意味が失われてなにやら不明瞭な線のかさなりになりはじめていた。意味を理解する間もなく次の文字を目が追うせいで前の文字が残ったまま次々に重なって、頭のなかは黒く塗りつぶされたみたいだった。

山登りは好きではないけれど、お寺や神社があればついてきてくれる妻を誘って出かける準備をはじめたのは午前の遅い時刻になってからだった。ひさしぶりに汗がでるような日差しが午後に向けていちだんと強まりつつあって、路面からは熱気が立ち上がっていた。

<img src="{{ '/assets/uploads/2018/06/DSC08843.jpg' | relative_url }}" class="alignnone wp-image-2436 size-full" height="533" alt="高尾山 1号路" width="800" srcset="{{ '/assets/uploads/2018/06/DSC08843.jpg' | relative_url }} 800w, {{ '/assets/uploads/2018/06/DSC08843-300x200.jpg' | relative_url }} 300w, {{ '/assets/uploads/2018/06/DSC08843-768x512.jpg' | relative_url }} 768w" sizes="(max-width: 800px) 100vw, 800px" /> 

連日の雨の中、晴れ間が出れば誰もみな同じことを考えるのだろう。高尾山口駅は沢山の人でにぎわっていた。山頂まで行列が続くというのは聞いたことがあったけれど、老若男女、普段着から上下ともに本格的な装備でそろえた人まで多種多様な人々が集っていた。

<img src="{{ '/assets/uploads/2018/06/DSC08842.jpg' | relative_url }}" class="alignnone wp-image-2430 size-full" height="533" alt="" width="800" srcset="{{ '/assets/uploads/2018/06/DSC08842.jpg' | relative_url }} 800w, {{ '/assets/uploads/2018/06/DSC08842-300x200.jpg' | relative_url }} 300w, {{ '/assets/uploads/2018/06/DSC08842-768x512.jpg' | relative_url }} 768w" sizes="(max-width: 800px) 100vw, 800px" /> 

中腹にある薬王院で御朱印をいただくため長い列に加わって歩く。  
道は舗装されて高低差もゆるやかなため歩きやすいのだが、妻が突然立ち止まり動かなくなった。買ったばかりの靴のせいで足が痛むのかと心配をしていると、彼女は目の前にある六根清浄石ぐるまを回しながら唄うような調子をつけて「さーんげさんげぇ（懺悔懺悔）、ろっこんしょーじょー（六根清浄）」と念仏を唱えた。突然のことに目を点にする私に妻は満足そうに満面の笑みを浮かべていた。

<img src="{{ '/assets/uploads/2018/06/DSC08856.jpg' | relative_url }}" class="alignnone size-full wp-image-2432" height="533" alt="薬王院" width="800" srcset="{{ '/assets/uploads/2018/06/DSC08856.jpg' | relative_url }} 800w, {{ '/assets/uploads/2018/06/DSC08856-300x200.jpg' | relative_url }} 300w, {{ '/assets/uploads/2018/06/DSC08856-768x512.jpg' | relative_url }} 768w" sizes="(max-width: 800px) 100vw, 800px" /> 

天平16年（744）、前年に東大寺の大仏建立の勧進を成功させ日本で初めて大僧正の位についた行基は、続いて聖武天皇の命により高尾山に東国鎮守の祈願寺として薬王院を開いた。5年後、行基が81歳で入滅すると薬王院は次第に荒廃していったが、約600年後の天授1年（1375）沙門俊源に再興された。現在の本尊である飯縄大権現は俊源が八千枚の護摩を焚いたすえに感得し奉祀したと云われる。

<img src="{{ '/assets/uploads/2018/06/DSC08878.jpg' | relative_url }}" class="alignnone wp-image-2429 size-full" height="533" alt="" width="800" srcset="{{ '/assets/uploads/2018/06/DSC08878.jpg' | relative_url }} 800w, {{ '/assets/uploads/2018/06/DSC08878-300x200.jpg' | relative_url }} 300w, {{ '/assets/uploads/2018/06/DSC08878-768x512.jpg' | relative_url }} 768w" sizes="(max-width: 800px) 100vw, 800px" /> 

薬王院でお参りをすませ、妻とふたり「懺悔懺悔、六根清浄」と念仏を繰り返しながら歩いているとやがて山頂にたどり着いた。

<img src="{{ '/assets/uploads/2018/06/DSC08865.jpg' | relative_url }}" class="alignnone wp-image-2435 size-full" height="533" alt="" width="800" srcset="{{ '/assets/uploads/2018/06/DSC08865.jpg' | relative_url }} 800w, {{ '/assets/uploads/2018/06/DSC08865-300x200.jpg' | relative_url }} 300w, {{ '/assets/uploads/2018/06/DSC08865-768x512.jpg' | relative_url }} 768w" sizes="(max-width: 800px) 100vw, 800px" /> 

山頂といってもいくつもの店があり山の上という感じはしないのだが、眼下には東京のビル群が小さく見えていて、朝に黒く塗りつぶされていた頭はすっかり元の通りのからっぽになっていた。

<img src="{{ '/assets/uploads/2018/06/DSC08890.jpg' | relative_url }}" class="alignnone size-full wp-image-2434" height="533" alt="高尾山 3号路" width="800" srcset="{{ '/assets/uploads/2018/06/DSC08890.jpg' | relative_url }} 800w, {{ '/assets/uploads/2018/06/DSC08890-300x200.jpg' | relative_url }} 300w, {{ '/assets/uploads/2018/06/DSC08890-768x512.jpg' | relative_url }} 768w" sizes="(max-width: 800px) 100vw, 800px" /> 

遅めの昼食にそばを食べて下山は静かな3号路を歩く。斜面にはあちこちに梅雨時の植物、ギンリョウソウが顔を出していた。

<img src="{{ '/assets/uploads/2018/06/DSC08885.jpg' | relative_url }}" class="alignnone wp-image-2433 size-full" height="533" alt="ギンリョウソウ" width="800" srcset="{{ '/assets/uploads/2018/06/DSC08885.jpg' | relative_url }} 800w, {{ '/assets/uploads/2018/06/DSC08885-300x200.jpg' | relative_url }} 300w, {{ '/assets/uploads/2018/06/DSC08885-768x512.jpg' | relative_url }} 768w" sizes="(max-width: 800px) 100vw, 800px" /> 

常緑樹の薄暗い木陰の中、葉緑素を持たないその花は仄白く亡霊のように不気味だった。
